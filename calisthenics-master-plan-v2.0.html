Two progressive strength systems running in parallel. Push-ups (upper body) and pistol squats (lower body) alternate — so you're always either building or recovering, but never both at once. Same 2-week block system, same recovery rules, completely different muscle groups.

How the Block System Works
Structure
Each progression = 4 blocks of 2 weeks. Each block = 6 sessions (3x per week). Complete a block then come back to Claude to generate the next one.
Why 3x Per Week
Muscle growth happens during REST. 48-72 hour recovery between sessions = body repairs stronger. More is not better. Rest days are when you actually get stronger.
Push-Up Progression (Upper Body)

Warm-up: Quick - LEG · Cool-down: post-push-up stretch (4 min) · Total: 16-22 min per session

Push-Up Blocks

Goal: Full floor push-ups by Block 4
Block 1 · Weeks 1-2
Foundation — Wall Push-Ups
  • Wall push-ups — 3 sets x 10-12 reps
  • Plank hold — 3 sets x 15-30 sec
  • Scapular push-ups — 2 sets x 8-10 reps
Complete — Mar 13
Block 2 · Weeks 3-4
Progression — Countertop Push-Ups
  • Countertop push-ups — 3 sets x 8-12 reps
  • Eccentric training — 3 sets x 5-8 reps (4-5 sec descent)
  • Scapular push-ups — 2 sets x 10-12 reps
  • Plank hold — 3 sets x 30-45 sec
In Progress — 5/6
Block 3 · Weeks 5-6
Advancement — Chair/Bench Push-Ups
  • Chair/bench push-ups — 3 sets x 8-12 reps
  • Eccentric training — 3 sets x 5-8 reps
  • Knee push-ups — 2 sets x 10-12 reps
  • Plank hold — 3 sets x 45-60 sec
Upcoming
Block 4 · Weeks 7-8
Mastery — Floor Push-Ups
  • Full floor push-ups — 3 sets x 5-10 reps
  • Eccentric floor push-ups — 3 sets x 3-5 reps
  • Knee push-ups (between sets) — as needed
  • Plank hold — 3 sets x 60 sec
Upcoming
Universal Form Guidelines ▼
  • Hands slightly wider than shoulders, weight distributed across entire palm
  • Elbows at 45° from body — "arrow" shape, not "T" shape
  • Core engaged — entire body one rigid plank, no sagging hips
  • Inhale on the way down, exhale on the way up. Never hold your breath.
  • Lower until chest nearly touches surface, push back to full extension
  • Scapular push-ups always before plank hold to preserve form quality
Pistol Squat Progression (Lower Body)

Warm-up: SFB or Quick + LEG · Cool-down: post-pistol squat stretch (4 min) · Total: ~25 min per session · Equipment: bodyweight, wall, chair, yoga blocks, jade mat

Pistol Squat Blocks

Goal: Full unassisted pistol squat by ~August 2026
Block 1 · Foundation
Base Strength
  • Bodyweight squats — 3 sets x 12-15 reps
  • Wall sit — 3 sets x 30-45 sec
  • Glute bridges — 3 sets x 12-15 reps
  • Step-ups — 2 sets x 10 reps each leg
Starting Apr 16
Block 2 · Single-Leg Work
Unilateral Strength
  • Bulgarian split squats — 3 sets x 8-10 reps each
  • Single-leg glute bridges — 3 sets x 10-12 reps each
  • Assisted single-leg squat (to chair) — 3 sets x 5-8 each
  • Wall sit — 3 sets x 45-60 sec
~June 2026
Block 3 · Depth and Control
Full Range of Motion
  • Eccentric single-leg squat — 3 sets x 5-6 reps each
  • Assisted pistol squat (yoga block/wall) — 3 sets x 5-8 each
  • Bulgarian split squats — 3 sets x 10-12 reps each
  • Single-leg wall sit — 3 sets x 20-30 sec each
~July 2026
Block 4 · The Pistol
Full Pistol Squat
  • Pistol squat attempts — 3 sets x 3-5 reps each
  • Eccentric pistol squat — 2 sets x 5 reps each
  • Bulgarian split squats — 2 sets x 12 reps (maintenance)
  • Wall sit — 2 sets x 60 sec (maintenance)
~August 2026
